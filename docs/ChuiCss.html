OS-Specific Themes

ChocolateChip-UI solves the problem of cross-platform app development by using themes to change the look and feel of an app to match the operating system it is running on. Because a theme is just CSS, it is easy to modify or extend a theme by adding new classes that you can use with existing widgets. If you are good with JavaScript, you might even craft your own widgets and style them with new styles. You can add the new styles to each theme.

Customize or Brand Your Theme

ChocolateChip-UI uses LESS to define the CSS styles. Just look for the file "colors.less" in side each theme folder. The theme LESS can be found at "src/theme." At the start of colors.less you'll find this: @appTint: #81cfeb;. By changing that color value and reprocess the less you'll get a theme with a different accent color from the default. Remember to change this value in the colors.less file for all three themes.

Download a Custom Theme

If you'd like to create a your own color theme, we have a custom red theme that you can download and use as the basis for yours. You'll just need to open each theme's "color.less" file and change the hex value of "@appTint" to the color you desire.
