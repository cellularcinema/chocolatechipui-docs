ChocolateChip-UI version 3.5.3

jQuery

Choose Your Source

ChocolateChip-UI is requires jQuery 2.0.3 or later. Proper functionality cannot be guaranteed for earlier versions. When using any jQuery plugins, please check that they were with jQuery 2.0.3 and later.

Setup

To build ChocolateChip-UI is as simple as typing "grunt. Open up your terminal, navigate to the downloaded repository. Run npm install. Then run grunt. This will build out a version of ChocolateChip-UI using a CDN version of jQuery version 2.0.3. If you want to use a different CDN or use a local copy, you just need to open up the file "package.json". Look for the value "jquery":: 

"jquery": {
   "url": "http://code.jquery.com/jquery-2.0.3.min.js"
},
Change the url value to what you need for your project. Then save and run grunt in the terminal and you'll have your project with all examples using the path you provided. For information about setting up Grunt for building, please see the project's README.
